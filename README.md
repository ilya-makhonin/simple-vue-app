# Vue application
The app is created on Vue.js framework. Node.js is used for a server side. MongoDB is used for saving data of users.

***

Technologies stack:

+ Vue.js
+ Node.js
+ Mongoose
+ CSS-in-JS library or material-design? 
+ Express.js

The project was created using Vue-cli
